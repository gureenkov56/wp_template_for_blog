== Description ==

Template for desiger or travel blog, where images is the most important.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in AirArt in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
5. Install the plugin "Social Icons Widget & Block by WPZOOM" for display social icons at the top of the site.
